# Student Guide

Use this folder when you are building a student project or need help during the
workshop.

## Start Here

1. [Student Handout](./handout.md) - the challenge, what to submit, and demo
   questions.
2. [Project Cards](./project-cards.md) - project ideas you can copy, combine, or
   adapt.
3. [Student AI Limits And Advice](./ai-limits.md) - how to use your app's
   server-side coach without wasting requests or exposing private data.
4. [AI Coding Prompts](./ai-coding-prompts.md) - focused prompts for planning,
   debugging, context building, testing, and responsible AI notes.
5. [Testing For Fast Feedback](./testing-fast-feedback.md) - why tests help you
   move faster, how to write manual test cases, and when to run smoke tests.
6. [Extend Soma](../extend-soma.md) - what you can change, what contracts to
   preserve, and how to test a remix.

## When You Are Stuck

| Problem | Use |
|---|---|
| I do not know what to build. | [Project Cards](./project-cards.md) |
| I need the workshop challenge. | [Student Handout](./handout.md) |
| The AI coach is failing or rate-limited. | [Student AI Limits And Advice](./ai-limits.md) |
| I need to build with local dummy data. | [Student AI Limits And Advice](./ai-limits.md#how-to-avoid-wasting-requests), [Workshop Lab B](../workshop/labs/README.md#lab-b-add-a-new-topic-pack) |
| I need to test my changes before demo. | [Testing For Fast Feedback](./testing-fast-feedback.md) |
| I need a good AI coding assistant prompt. | [AI Coding Prompts](./ai-coding-prompts.md) |
| I need to connect a concept to a file. | [Concept To Code Map](../workshop/concept-to-code-map.md) |
| I need hands-on practice. | [Workshop Labs](../workshop/labs/README.md) |
| I want to extend or remix Soma. | [Extend Soma](../extend-soma.md) |
| A mentor asked me to test a real key. | [Gemini Key Setup](../gemini-key-setup.md) |
| A mentor asked me to deploy. | [Deploy To Vercel](../deploy-vercel.md) |

## Safe Student Rule

Use dummy learning data. Do not enter names, school names, marks, phone
numbers, addresses, or private records into the app or an AI coding assistant.
